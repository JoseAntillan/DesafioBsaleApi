Introducción

Esta documentación lo ayudará a familiarizarse con los recursos de la API "Bsale" y entregarle todo lo necesario para comenzar con sus solicitudes.

URL base

https://desafiobsaleapi.onrender.com/
Todas las solicitudes son solicitudes GET. Todas las respuestas devolverán datos en formato JSON.

En esta API podemos acceder a 2 recursos;

  1. Productos
  2. Categorias

Productos Endpoint

Atributos de productos que devuelve la API

Atributo Tipo Descripcion
id int Identificador único del producto
name String Nombre del producto
url_image String URL de la imagen asociada al producto
price Float Precio de venta del producto
discount int Porcentaje de descuento del producto
Ccategory_id int Identificador de la categoría
Ccategory_name String nombre de la categoría del producto

Obtener todos los productos:

 Para obtener una lista con todos los productos:

/productos

Obtener todos los productos que esten en una categoria:

/categorias/idcategoria/listarproductos

ejemplo para obtener todos los productos de la categoria 5:

/categorias/5/listarproductos

Buscar productos:

 Para obtener una lista con todos los productos que contienen la palabra buscada en el nombre:

/productos?busqueda=

ejemplo para buscar papas:

/productos?busqueda=papas

Categorias Endpoint

Atributos de categoria que devuelve la API

Atributo Tipo Descripcion
id int Identificador único de la categoria
name String Nombre de la categoria
cantidad_productos int Cantidad de productos que existen en la categoria

Obtener todas las categorias:

 Para obtener una lista con todas las categorias:

/categorias
